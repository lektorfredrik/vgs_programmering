Tegne grafer med matplotlib#

Her skal vi ta en titt på hvordan vi kan tegne grafer med matplotlib. Vi trenger:

  1. Importere matplotlib.pyplot as plt

  2. En liste med x-verdier og en liste med y-verdier (typisk er y-verdiene funksjonsverdier).

  3. Bruke plt.plot(x, y) for å tegne grafen.

Eksempler#

Eksempel 1#

Tenk deg at vi ønsker å tegne grafen til

\[ f(x) = x^2, \qquad x \in [-2 , 2]. \]

Dette kan vi gjøre slik:

import matplotlib.pyplot as plt # bibliotek for tegne grafer
import numpy as np # mattebibliotek

# Lage funksjonsverdier
x = np.linspace(-2, 2, 1001) # lager x-verdier mellom -2 og 2
y = x**2 # lager y-verdier

# Tegne grafen
plt.plot(x, y) # tegner grafen
plt.show() # Viser grafen

Eksempel 2#

Tenk deg at vi ønsker å tegne grafen til

\[ f(x) = x^3 - 2x + 1, \qquad x \in [-1 , 1]. \]

Denne gangen ønsker vi å pynte litt mer på grafen. Vi ønsker å:

  • Sette navn på aksene. Dette gjøres med

    • plt.xlabel("tekst") for å sette navn på \(x\)-aksen.

    • plt.ylabel("tekst") for å sette navn på \(y\)-aksen.

  • Legge til rutenett. Dette gjøres med plt.grid(True).

import matplotlib.pyplot as plt
import numpy as np

# Lage funksjonsverdiene
x = np.linspace(-1, 1, 1000) # lager x-verdiene
y = x**3 - 2 * x + 1 # regner ut y-verdiene

# Tegne grafen
plt.plot(x, y) # tegner grafen
plt.xlabel("x") # navn på x-akse
plt.ylabel("y") # navn på y-akse
plt.grid(True) # lager rutenett
plt.show() # viser grafen

Eksempel 3#

La oss tegne grafen til

\[ g(x) = x^2 e^{-x}, \qquad x \in [0 , 4]. \]

Denne gangen ønsker vi å:

  • Tegne grafen i rødt. Dette gjøres med plt.plot(x, y, color="red").

  • Legge til en tittel. Dette gjøres med plt.title("tekst").

I tillegg tar vi med navn på aksene og rutenett som i eksempel 2.

import matplotlib.pyplot as plt
import numpy as np

# Lage funksjonsverdiene
x = np.linspace(0, 4, 1000) # lager x-verdiene. 1000 punkter mellom 0 og 4
y = x**2 * np.exp(-x) # regner ut funksjonsverdiene til x^2 * e^{-x}

# Tegne grafen
plt.plot(x, y, color="red") # tegner grafen i rødt
plt.xlabel("$x$") # navn på x-akse. `$x$` gir mattenotasjon
plt.ylabel("$g(x)$") # navn på y-akse. `$g(x)$` tegner det med mattenotasjon
plt.grid(True) # lager rutenett
plt.title("En tittel på grafen") # tittel på grafen
plt.show() # viser grafen

Eksempel 4#

La oss tegne grafene til

\[ f(x) = x^2, \qquad x \in [-1 , 1], \qquad \text{og} \qquad g(x) = x^3 - 2x + 1, \qquad x \in [-1 , 1], \]

i samme figur.

Her ønsker vi å legge til navn på grafene så vi vet hvilke av de som er hvilke. Dette gjøres med å legge til label="navn" i plt.plot(x, y, label="navn"). Deretter må vi legge til plt.legend() for å få med navnene på grafene.

import matplotlib.pyplot as plt # bibliotek for tegne grafer
import numpy as np # mattebibliotek

# Lage funksjonsverdier
x = np.linspace(-1, 1, 1001) # lager x-verdier mellom -1 og 1
f = x**2
g = x**3 - 2 * x + 1

# Tegne grafen
plt.plot(x, f, label="$f(x)=x^2$") # tegner grafen til f
plt.plot(x, g, label="$g(x)=x^3-2x+1$") # tegner grafen til g
plt.xlabel("$x$") # navn på x-akse
plt.ylabel("$y$") # navn på y-akse
plt.grid(True) # lager rutenett
plt.legend() # viser navnene på grafene
plt.show() # viser grafen

Påminnelse: regne ut tangenter

Vi kan finne tangenten til \(f\) i punktet \(x = a\) ved å bruke formelen

\[ T(x) = f'(a)(x - a) + f(a). \]
Påminnelse: produktregelen for derivasjon

\[ f'(x) = g'(x)h(x) + g(x)h'(x). \]

Den deriverte av \(f(x)\) er gitt ved

\[ \frac{df}{dx} = f'(x) = 2xe^{-x^2} - 2x^3e^{-x^2} = 2xe^{-x^2}(1 - x^2). \]

import matplotlib.pyplot as plt
import numpy as np

def f(x):
    return x**2 * np.exp(-x**2)

def dfdx(x):
    return 2 * x * np.exp(-x**2) * (1 - x**2)


# Lage funksjonsverdier
x = np.linspace(-1, 2, 1001) # lager x-verdier mellom -1 og 2
y = f(x)
T = dfdx(x=0.5) * (x - 0.5) + f(x=0.5)


plt.plot(x, y, label="$f(x)=xe^{-x}$", color="red") # tegner grafen til f
plt.plot(x, T, label="Tangent", color="blue") # tegner tangenten
plt.xlabel("$x$") # navn på x-akse
