if-else tester#

Læringsmål#

  • Forstå hva en if-else-test kan brukes til og hvordan det bestemmer flyten i en kodeblokk.

  • Forstå hvordan man setter opp ulike betingelser i Python.

  • Kunne lese og skrive kode med if-elif-else-test.

  • Kunne lese og skrive kode med mer enn én elif-blokk.

Introduksjon#

Et sentralt verktøy i programmering er if-else tester. De lar oss kjøre deler av programmet hvis en betingelse er oppfylt, og kjøre en annen hvis den ikke er det.

Enkle if-else tester#

En enkel if-else test vil har formen

1if betingelse:
2    # Kjør denne koden hvis betingelse er True
3else:
4    # Kjør denne koden hvis betingelse er False

Vi kaller den første delen for if-blokken og den andre for else-blokken.

Vi kan visualisere prosessen med et flytdiagram:

flowchart TD A[Start] --> B[if-else test] --> C(Sjekk om betingelse er sann) C -- Sann --> D[Utfør handling når betingelsen er sann] C -- Usann --> E[Utfør handling når betingelsen er usann] D --> F[Ferdig] E --> F

La oss se på et eksempel:

1a = 1
2if a >= 0:
3    print("a er større enn eller lik null.")
4else:
5    print("a er negativ eller null.")

a er større enn eller lik null.

Det vi kan lære av dette er at a >= 0 er en betingelse som kan være True eller False. Hvis den er True kjører vi koden i den første blokken, hvis den er False kjører vi koden i den andre blokken. Her er a = 1, så a >= 0 er True, og vi får utskriften a er større enn eller lik null.

Vi kan printe ut betingelsen a >= 0 for å se at den faktisk evalueres til True:

1print(a >= 0)

True

La oss si vi setter a = -1. Hva blir da utskriften? Hvorfor?

Betingelser i Python#

Python har innebygde måter å skrive betingelser. Dette lar oss sjekke om en betingelse er oppfylt eller ikke. Resultatet av en slik sammenlikning er alltid enten at betingelsen er sann som gir True, eller at den er usann som gir False.

I Python kan vi sette opp betingelser for på flere måter. I tabell 1 vises de vanligste sammenlikningsoperatorene.

Tabell 1: Tabellen viser de vanligste logiske operatorene. Som eksempel har vi satt a = 1 og b = -2 for å vise hva resultatet av operasjonen blir.

Underveisoppgave 2#

Hva blir resultatet av 1 != 2?

Løsningsforslag

Siden 1 er ulik 2, så vil 1 != 2 være True.

Underveisoppgave 3#

La r = -5 og s = 10. Finn en betingelse som kan settes på r og s slik at resultatet blir False?

Løsningsforslag

Her er det flere muligheter. Noen eksempler er

  1. r == s gir False fordi r og s har ulik verdi.

  2. r > s gir False fordi r er mindre enn s.

  3. r >= s gir False fordi r er mindre enn s.

  4. r + s != 5 gir False for r + s er lik 5.

Underveisoppgave 4#

Lag en variabel med navn res med en verdi slik at følgende kodesnutt kjører if-blokken. Hva er en passende tekst å skrive ut til brukeren i if-blokken?

Ta utgangspunkt i kodeskallet under. Du må fylle inn der det står NotImplemented.

1res = NotImplemented
2
3if res >= -2:
4    print("{res} er større enn eller lik -2")
5else:
6    print(f"{res} er mindre enn -2.")

Løsningsforslag

I if-testen, så sjekker vi om res er større enn eller lik -2. Hvis vi ønsker at if-blokken skal kjøres, så må res være større enn eller lik -2 fordi vi trenger at betingelsen skal bli True. Vi kan med andre ord sette res til en tilfeldig verdi som er større enn eller lik -2.

Det vil si res = -2 vil gi True, men det vil alle andre verdier som er større enn -2 også. Så som et eksempel kan koden se slik ut:

res = -1

if res >= -2:
    print(f"{res} er større enn eller lik -2.")
else:
    print(f"{res} er mindre enn -2.")

Sjekke flere betingelser#

Når vi har flere betingelser vi ønsker å sjekke, gir Python følgende muligheter:

  • Vi kan bruke and for å sjekke om flere betingelser er oppfylt samtidig.

  • Vi kan bruke or for å sjekke om minst én av flere betingelser er oppfylt.

  • Vi kan sette sammen betingelser av typen \(a \leq b < c\).

Vi kan kjede sammen flere betingelser i kombinasjon med enten and eller or.

Eksempel 1:#

Tenk deg at vi skal sjekke om \(a\) både er større enn \(b\) og mindre enn \(c\). Matematisk betyr dette at \(b < a < c\) eller at \(b < a\) og \(a < c\). I Python kan vi skrive dette som

b < a < c # Alternativ 1

b < a and a < c # Alternativ 2

Alternativ 1 gjør akkurat det samme som alternativ 2, men and er gjemt bort ved at Python sjekker betingelsene b < a og a < c parvis og gir True bare hvis begge er oppfylt samtidig.

Eksempel 2:#

Tenk deg at vi ønsker å sjekke om \(a\) er mindre enn \(b\) eller \(c\). Dette er effektivt det samme som å si at \(a < b\) eller \(a < c\). Vi kan skrive en samlet betingelse for de to med or mellom dem:

a < b or a < c

Underveisoppgave 5#

Dersom vi har at a = 1, b = -3 og c = 3, hva blir resultatet av a < b and a < c?

Løsningsforslag

Siden a < b er False og a < c er True, så vil a < b and a < c være False fordi begge betingelser ikke er oppfylt samtidig.

Underveisoppgave 6#

Dersom vi har a = 1, b = 2 og c = -1, hva blir resultatet av a < b or a < c?

Løsningsforslag

Siden a < b er True vil a < b or a < c være True siden én av de to betingelsene er sanne.

Underveisoppgave 7#

Dersom vi har a = 1, b = 2 og c = -1,

  1. Hva blir resultatet av a < b < c?

  2. Kan du skrive om betingelsen til to betingelser ved å bruke and eller or?

Løsningsforslag

  1. Resultatet av a < b < c blir False siden a < b er True og b < c er False.

  2. Vi kan skrive om a < b < c til a < b and b < c siden vi må sjekke begge betingelser samtidig.

if-elif-else-tester#

Ofte ønsker vi å skrive kode der hvilken del av koden vi kjører ikke bare er to-delt, men det finnes flere betingelser som gjør at andre deler av koden skal kjøres. Da kan vi bruke if-elif-else-tester.

Den generelle strukturen til en if-elif-else-test er

1if betingelse1: # Sjekk om betingelse 1 er sann
2    # Kjør kode hvis betingelse1 er sann
3elif betingelse2: # Hvis ikke betingelse1 er sann, sjekk om betingelse2 er sann
4    # Kjøre kode hvis betingelse2 er sann
5else: # Hvis ingen av betingelse1 og betingelse2 er sanne
6    # Kjør koden som står her

  • På linje 1 sjekker vi om betingelse1 er sann. Hvis den er det, så kjører vi koden som står i if-blokken.

  • På linje 3 sjekker vi om betingelse2 er sann hvis betingelse1 ikke var det.

  • På linje 5 kjører vi koden som står i else-blokken hvis verken betingelse1 eller betingelse2 er sanne.

Flytdiagrammet under vises hvordan en if-elif-else-test fungerer.

flowchart TD A[Start] --> B[if-elif-else test] --> C[Sjekk om if-betingelsen er sann] C -- Sann --> D(if-blokk kjøres) --> G[Ferdig] C -- Usann --> E[Sjekk om elif-betingelsen er sann] E -- Sann --> F(elif-blokk kjøres) --> G[Ferdig] E -- Usann --> H[else-blokk kjøres] --> G[Ferdig]

Eksempel: moms på varer#

Når vi betaler for ulike tjeneser i Norge, er det ulike avgift som betales avhengig av varen.

  • For matvarer er det 15% moms.

  • For reiser er det 12% moms.

  • For generelle varer er det 25% moms.

La oss se på et program som velger ut moms avhengig av hvilken vare det er:

 1varetype = "reise"
 2
 3if varetype == "mat":
 4    moms = 15 
 5elif varetype == "reise":
 6    moms = 12 
 7else:
 8    moms = 25
 9
10print(f"{moms = } prosent på varen din av type `{varetype}`.")

moms = 12 prosent på varen din av type `reise`.

Gjennomgang av kodeeksempelet

  • På linje 1 setter vi varetype = "reise".

  • På linje 3 sjekker programmet om varetype er lik "mat". Siden den ikke var det, hopper vi over if-blokken av koden.

  • På linje 5 sjekker koden om varetype er lik "reise". Det er den, så elif-blokken kjøres og vi setter moms = 12.

  • På linje 7 starter else-blokken. Men siden én av betingelsene var sann, så hopper vi over denne blokken.

Med et flytdiagram, kan vi visualisere koden slik:

flowchart TD A[Start: velg en varetype] --> B{if-elif-else-test} B --> C[Sjekk om varetype er mat] C -- Ja --> D[Sett moms til 15%] C -- Nei --> E[Sjekk om varetype er reise] E -- Ja --> F[Sett moms til 12%] E -- Nei --> G[Sett moms til 25%] D --> H[Skriv ut verdien] F --> H[Skriv ut verdien] G --> H[Skriv ut verdien] H --> I[Ferdig]

Underveisoppgave 8#

Hva blir verdien til y i koden under?

1x = 2.5
2
3if x < 0:
4    y = 1
5elif x < 1:
6    y = 2
7else:
8    y = 3

Løsningsforslag

Her er x sin verdi 2.5. Følger vi koden gjennom, så skjer følgende:

  • På linje 3 sjekker vi om x er mindre enn 0. Det er den ikke, så vi går videre til elif-blokken.

  • På linje 5 sjekker vi om x er mindre enn 1. Det er den heller ikke, så vi går videre til else-blokken.

  • Dermed setter vi y = 3 på linje 7.

Verdien til y blir derfor 3.

Underveisoppgave 9#

Tenk deg at du har den uferdige koden

1x = 0.5

Vi ønsker at elif-blokken skal kjøres hvis x er større enn 0 og mindre enn 1. Derfor er den riktig betingelsen 0 < x < 1 eller x > 0 and x < 1

Oppgave 1#

Lag et program som spør brukeren om alderen deres med input-funksjonen. Hvis alderen er under 18, skal programmet skrive ut Du er ikke myndig. Hvis alderen er 18 eller over, skal programmet skrive ut Du er myndig.

Du kan ta utgangspunkt i kodeskallet under. Du må fylle inn der det står NotImplemented.

1alder = int(input("Hvor gammel er du? "))
2
3if NotImplemented:
4    print(NotImplemented)
5else:
6    print(NotImplemented)

Løsningsforslag

To måter å løse dette på:

alder = int(input("Hvor gammel er du? "))

if alder < 18:
    print("Du er ikke myndig.")
else:
    print("Du er myndig.")

alder = int(input("Hvor gammel er du? "))

if alder >= 18:
    print("Du er myndig.")
else:
    print("Du er ikke myndig.")

Oppgave 2#

Lag et program som sammenlikner to heltall og skriver ut hvilket som er størst, og hvilken verdi det har. Det ene tallet skal skrives inn av brukeren med input-funksjonen.

Du kan ta utgangspunkt i kodeskallet under. Du må fylle inn der det står NotImplemented.

1x = 2
2y = int(input("Skriv inn et tall: "))
3
4if NotImplemented:
5    print(NotImplemented)
6elif NotImplemented:
7    print("Tallene har samme verdi")
8else:
9    print(NotImplemented)

Løsningsforslag

Her er det flere muligheter her, men en løsning er:

x = 2
y = int(input("Skriv inn et tall: "))

if y > x:
    print(f"{y = } er det største tallet.")
elif x == y:
    print("Tallene har samme verdi")
else:
    print(f"{x = } er det største tallet.")

Sett en betingelse på x slik at y får verdien 2 i koden under:

 1x = 2
 2
 3if 0 < x < 2:
 4    y = -2 * x
 5elif NotImplemented:
 6    y = 2

 8    y = x**2
 9

x = 2

Oppgave 4#

La \(a\), \(b\) og \(c\) være lengdene til sidene i en trekant. Lag et program som

  • Tar inn a, b og c fra brukeren med input-funksjonen,

  • Sjekker om trekanten er rettvinklet eller ikke.

  • Test ut programmet med:

    • \(a = 3\), \(b = 4\) og \(c = 5\),

    • \(a = 4\), \(b = 3\) og \(c = 5\),

    • \(a = 1\), \(b = 2\) og \(c = 3\).

Hvis trekanten er rettvinklet, skal programmet printe ut en melding som sier at den er rettvinklet. Hvis ikke skal den printe ut en melding der den sier at trekanten ikke er rettvinklet.

Kodehint 1 - input-funksjonen

Du kan bruke følgende linjer på toppen av programmet ditt hvis du er usikker på hvordan input-funksjonen fungerer:

a = float(input("Fyll inn lengden til side 1 i trekanten: "))
b = float(input("Fyll inn lengden til side 2 i trekanten: "))
c = float(input("Fyll inn lengden til side 3 i trekanten: "))

Kodehint 2

For å sjekke om en trekant er rettvinklet, kan du bruke Pytagoras’ setning. Sidene i seg selv har ingen spesiell rekkefølge her, så du må sjekke alle kombinasjoner:

  1. \(a^2 = b^2 + c^2\)

  2. \(b^2 = a^2 + c^2\)

  3. \(c^2 = a^2 + b^2\)

Hvis én av disse likningene er oppfylt, er trekanten rettvinklet. Hvis ingen av dem er oppfylt, er ikke trekanten rettvinklet.

Løsningsforslag

a = float(input("Fyll inn lengden til side 1 i trekanten: "))
b = float(input("Fyll inn lengden til side 2 i trekanten: "))
c = float(input("Fyll inn lengden til side 3 i trekanten: "))

# Sjekk a^2 = b^2 + c^2
if a**2 == b**2 + c**2: 
    print("Trekanten er rettvinklet")

# Sjekk b^2 = a^2 + c^2
elif b**2 == a**2 + c**2:
    print("Trekanten er rettvinklet")

# Sjekk c^2 = a^2 + b^2    
elif c**2 == a**2 + b**2:
    print("Trekanten er rettvinklet")

# Hva skal skrives ut hvis ingen av testene er sanne?
else:
    print("Trekanten er ikke rettvinklet")

For de tre tilfellene skal du få:

  • \(a = 3\), \(b = 4\) og \(c = 5\) skal gi “rettvinklet”.

  • \(a = 4\), \(b = 3\) og \(c = 5\) skal gi “rettvinklet”.

  • \(a = 1\), \(b = 2\) og \(c = 3\) skal gi “ikke rettvinklet”.

Oppgave 5#

La \(a\), \(b\) og \(c\) være sidene i en trekant. Lag et program som

  • Tar inn verdiene til \(a\), \(b\) og \(c\) fra brukeren med input-funksjonen.

  • Sjekker om trekanten er likebeint, likesidet eller ingen av delene.

  • Test ut programmet med følgende kombinasjoner av \(a\), \(b\) og \(c\):

    • \(a = 1\), \(b = 2\) og \(c = 3\),

    • \(a = 2\), \(b = 2\) og \(c = 3\),

    • \(a = 3\), \(b = 3\) og \(c = 3\).

Programmet skal skrive ut en melding som sier om trekanten er likesidet, likebeint eller ingen av delene.

Hva er en likebeint og en likesidet trekant?

  • En likebeint trekant er en trekant der to av sidene er like lange. Du må derfor sjekke om minst to av sidene er like.

  • Dersom alle sidene er like lange, er trekanten likesidet.

Kodehint: Hvordan sjekke flere betingelser samtidig

Her kommer du til å få nytte av å kunne sjekke flere betingelser samtidig. Hvis vi har en betingelse1 og en betingelse2, så kan vi:

  • Sjekke om begge er oppfylt samtidig ved å skrive betingelse1 and betingelse2. Man plasserer altså en and mellom dem for å sjekke om både betingelse1 og betingelse2 er oppfylt samtidig.

  • Sjekke om minst én av betingelsene er oppfylt ved å skrive betingelse1 or betingelse2. Man plasserer altså or mellom dem for å sjekke om enten betingelse1 eller betingelse2 er oppfylt, men ikke nødvendigvis begge to samtidig.

Løsningsforslag

Det er flere løsninger, men her er én mulighet:

a = float(input("Fyll inn lengden til side 1 i trekanten: "))
b = float(input("Fyll inn lengden til side 2 i trekanten: "))
c = float(input("Fyll inn lengden til side 3 i trekanten: "))

# Sjekk om trekanten er likesidet
if a == b and b == c:
    print("Trekanten er likesidet.")

# Sjekk om trekanten er likebeint
elif a == b or a == c or b == c:
    print("Trekanten er likebeint.")

# Hva bør du skrive ut dersom ingen av testene er sanne?
else:
    print("Trekanten er hverken likesidet eller likebeint.")

For de tre tilfelle skal du få:

  • \(a = 1\), \(b = 2\) og \(c = 3\) skal gi “hverken likesidet eller likebeint”.

  • \(a = 2\), \(b = 2\) og \(c = 3\) skal gi “likebeint”.

  • \(a = 3\), \(b = 3\) og \(c = 3\) skal gi “likesidet”.
